# Question 1

Share one technical concept that you developed greater mastery over in this project. Demonstrate how you understand that concept by sharing your mental model of the concept. Then, show how you used that concept in your project.

I mastered organizing the layout of my containers quicker and cleaner. Before I didn't use div or sections tags which made it hard to move my elements in the way I wanted. I also wasn't in the habit of using class or id selectors. I didn't understand their importance before but now I use them in most of my code and I feel more confident in designing my website.

# Question 3

How did you leverage AI to assist your development of this project?

I asked AI to teach me how to implement a dropdown feature that shows up when the screen is less than 768 px. Instead of adjusting the layout of my navigation bar I made it so that it would turn into a single word called navigation that opens to the options when you hover over it.
